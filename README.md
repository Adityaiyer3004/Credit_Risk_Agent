# Credit Risk Assessment Agent

Autonomous agent that pulls live UK company data from Companies House, runs a 
multi-factor risk scoring engine, and generates institutional-grade credit risk 
reports using Groq's Llama 3.1 — in under 3 seconds per company.

## The problem

Credit risk assessment on UK SMEs is manual, slow, and inconsistent. 
Analysts spend hours cross-referencing Companies House filings, insolvency 
records, and charge registrations before writing a report. This agent does 
it in seconds.

## How it works

1. **Data ingestion** — fetches live company profile, filing history, officer 
   data, insolvency records, and registered charges directly from Companies 
   House API
2. **Risk scoring** — multi-factor engine scoring company age, filing recency, 
   director count, insolvency flags, and sector risk (SIC codes) into a 
   1-100 credit score
3. **Legal enrichment** — checks insolvency and charge registrations via 
   Companies House endpoints, flags dissolved or liquidated companies
4. **LLM report generation** — structured 4-part credit report (executive 
   summary, strengths, risks, final rating) generated by Groq Llama 3.1 
   at sub-second latency
5. **Dashboard** — clean Streamlit UI with risk badges, company profile 
   cards, and an AI-generated report

## Tech stack

- **FastAPI** — REST API backend (`/api/analyze`)
- **Streamlit** — frontend dashboard
- **Groq + Llama 3.1** — low-latency LLM inference for report generation
- **LangChain** — LLM orchestration layer
- **Companies House API** — live UK company data (profile, insolvency, charges)
- **Python** — risk engine, data ingestion, orchestration

## Architecture

GET /api/analyze?company_number=XXXXXXXX
│
├── comp_house_ingestor.py   → live company profile
├── gazette_scraper.py       → insolvency + charge records
│
▼
risk_engine.py                   → credit score (1-100) + risk level
│
├── report_builder.py       → structured baseline report
├── llm_reporter.py         → Groq Llama 3.1 AI report
│
▼
orchestrator.py → FastAPI → Streamlit dashboard

## Risk scoring model

| Factor | Score impact |
|--------|-------------|
| Company inactive | -40 |
| Age < 2 years | -20 |
| No accounts filed | -15 |
| Accounts > 2 years old | -20 |
| No active directors | -20 |
| Insolvency record | -25 |
| Registered charges | -10 |
| High-risk SIC sector | -2 |

Final score → Low Risk (80+) / Medium Risk (50-79) / High Risk (<50)
